This solution is ideal for you use antispam in your systems.

Steps to run this demo:

In Windows, copy the DLLs "freetype-6.dll" and "zlib1.dll" from "lazsolutions\media\lib\win32" and font "LiberationSerif-Regular.ttf" from "lazsolutions\media\font" to this folder, and try run the demo again.

In Linux, copy the font "LiberationSerif-Regular.ttf" to this folder, and try run the demo again.

Enjoy!
